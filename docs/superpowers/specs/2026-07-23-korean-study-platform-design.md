# 국어 학습 플랫폼 설계

## 목표

기존 영어 학습 플랫폼과 독립된 Hexo 기반 국어 학습 사이트를 만든다. 초등·중등·고등 경로를 제공하고, 고등은 2028 수능 국어 공통 체계를 중심으로 안내한다.

## 사용자와 핵심 흐름

- 학생은 첫 화면에서 초등, 중등, 고등 2028 중 하나를 선택한다.
- 학년군 화면에서 주제 필터를 누르고 순서형 레슨 목록에서 한 레슨을 시작한다.
- 레슨은 읽기 또는 작품 소개, 핵심 개념, 확인 질문, 표현 활동, 다음 학습 순서로 끝난다.
- 보호자·교사는 같은 정적 URL을 공유해 학습 경로를 안내할 수 있다.

## 정보구조

| Route | Purpose |
|---|---|
| `/` | 세 학년군과 오늘의 학습 진입점 |
| `/elementary/` | 읽기 기초, 어휘, 이야기 이해 |
| `/middle/` | 문학, 독서, 문법, 글쓰기 |
| `/high-2028/` | 화법과 언어, 독서와 작문, 문학 |
| `/library/` | 시대·갈래별 국어국문학 자료실 |
| `/design-system/` | 구현 전 primitive showcase |

## 콘텐츠 모델

각 레슨 markdown front matter는 `level`, `subject`, `sequence`, `study_time`, `status`, `description`을 가진다. 본문은 기존 교재를 복제하지 않고 자체 해설과 적법한 인용·공개 자료만 사용한다.

## 2028 고등 국어 기준

고등 허브는 선택과목 분기가 아닌 `화법과 언어`, `독서와 작문`, `문학` 공통 경로를 사용한다. 수능 절대평가 등 확정되지 않은 예측은 사이트의 사실 정보로 표시하지 않는다.

## 디자인

`DESIGN.md`의 따뜻한 종이·짙은 잉크·소나무 녹색·절제된 금색 토큰을 따른다. 상단 탐색, 주제 필터, 순서형 레슨 행, 정적 학습 글이라는 제품 구성을 계승하되 다른 사이트의 브랜드, 문구, 이미지, 소스는 사용하지 않는다.

## 비목표

- 로그인, 저장 진도, 결제, 교사 대시보드
- 저작권이 불명확한 교재나 긴 작품 원문 복제
- 특정 교육기관·브랜드의 명칭 또는 시각 자산 재사용

## 검증 기준

- Hexo 정적 생성이 성공한다.
- 생성 HTML에 세 학년군, 2028 고등의 세 공통 영역, 최소 세 개의 레슨 경로가 존재한다.
- 필터 제어는 native button과 `aria-pressed`를 사용한다.
- 375px, 768px, 1280px에서 콘텐츠가 한 열로 읽히거나 적절하게 재배치되고 가로 스크롤이 없다.
