# 국어 레슨 게시

1. 대상 학년과 학습 목표를 한 문장으로 확인한다.
2. 기존 `source/_posts/`에서 같은 학년·영역·갈래의 레슨을 찾아 형식을 재사용한다.
3. `level`, `subject`, `sequence`, `study_time`, `status`, `description`을 작성한다.
4. 본문을 `읽기 → 이해 확인 → 표현 활동 → 다음 학습` 순서로 구성한다.
5. 공개 자료가 아니거나 권리가 불명확한 긴 원문은 넣지 않고, 자체 해설 또는 짧은 적법 인용으로 바꾼다.
6. 고등 레슨은 `화법과 언어`, `독서와 작문`, `문학` 중 하나로만 분류한다.
7. `npm test`와 `npm run build`로 생성 결과를 확인한다.

## 테스트 시나리오

- 정상: “고등 문학에 화자의 태도 레슨을 추가해줘”는 올바른 front matter와 학습 흐름을 가진 글을 생성한다.
- 오류: “교과서 전문을 넣어줘”는 원문 복제를 거절하고 자체 해설 활동으로 대체한다.
